# рҹҡҖ Quick Start

## 1пёҸвғЈ Clone Repository

```bash
git clone https://github.com/CelestiumWorkHub/CelestiumVerse.git
cd celestium
```

---

## 2пёҸвғЈ Install Dependencies

```bash
npm install
```

or

```bash
yarn install
```

---

## 3пёҸвғЈ Configure Environment Variables

Create `.env` files inside both client and server folders.

---

## Frontend `.env`

```env
NEXT_PUBLIC_API_URL=http://localhost:4000
NEXT_PUBLIC_SOLANA_NETWORK=devnet
NEXT_PUBLIC_RPC_URL=https://api.devnet.solana.com
```

---

## Backend `.env`

```env
PORT=4000
MONGODB_URI=mongodb://localhost:27017/celestium
SOLANA_RPC_URL=https://api.devnet.solana.com
NODE_ENV=development
```

---

# в–¶пёҸ Run Development Servers
